# Contributing

Thank you for your interest in contributing to this project.

Community contributions help improve the software and are greatly appreciated.

## Before You Contribute

Please discuss significant changes through an issue before investing substantial effort in implementation.

For small fixes, documentation improvements, and bug fixes, feel free to open a pull request directly.

## Coding Standards

Please:

* Follow the existing coding style.
* Keep pull requests focused and reasonably small.
* Write clear commit messages.
* Include tests where appropriate.
* Update documentation when necessary.
